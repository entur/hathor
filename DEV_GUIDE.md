# Hathor Developer Guide

Hathor is a fork of **[Inanna](https://github.com/entur/inanna)** — Entur's React 19 + Vite + MUI v7 data-registry starter.

Developer guidance for extending this codebase — adding entity views and data-table
pages, modelling new domains, working with the Generic Data View pattern — lives in
the global **`inanna-fork`** skill. Run it in *extend mode* against this
already-forked tree. The skill is the maintained, Entur-wide source of truth; this
guide used to duplicate it and drifted out of date, so it was reduced to this pointer.
